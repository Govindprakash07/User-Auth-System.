# User Authentication System 🔑  

A basic authentication system using Node.js, Express, MongoDB, and JWT.  

## Features  
✔️ User **registration and login**  
✔️ **Hashed passwords** using bcrypt  
✔️ **JWT-based authentication**  

## Technologies Used  
- Node.js  
- Express.js  
- MongoDB  
- bcrypt.js  
- jsonwebtoken (JWT)
